### OUTPUT FILES (Generated by Backtest)

```
📊 execution_log_v3.csv
   ├─ timestamp: When trade executed
   ├─ day: Which day
   ├─ product: INTARIAN or ASH
   ├─ side: BUY or SELL
   ├─ price: Execution price
   ├─ quantity: Shares filled
   ├─ position_qty: Position after trade
   ├─ position_cost: Average cost basis
   └─ realized_pnl: Profit/loss on closed positions
   
   EXAMPLE:
   4,-2,INTARIAN_PEPPER_ROOT,BUY,9999.69,3.93,3.93,9999.69,0.0
   8,-2,INTARIAN_PEPPER_ROOT,SELL,10005.00,2.0,1.93,9999.69,11.34

📊 results_v3.json
   ├─ total_pnl: Total profit/loss
   ├─ realized_pnl: Closed positions
   ├─ unrealized_pnl: Open positions
   ├─ intarian_pnl: Product breakdown
   ├─ ash_pnl: Product breakdown
   ├─ total_trades: Number of trades
   ├─ peak_equity: Highest equity
   ├─ max_drawdown: Worst loss
   ├─ sharpe_ratio: Risk-adjusted return
   └─ final_*_qty: Ending positions

   EXAMPLE OUTPUT:
   {
     "total_pnl": 15818.31,
     "sharpe_ratio": 21.98,
     "max_drawdown": 6.51,
     "total_trades": 98
   }
```

### Scenario 3: Grid Search Optimization

```python
import itertools

results_list = []

for buy_thresh, sell_thresh in itertools.product(
    [0.55, 0.60, 0.65, 0.70],
    [0.30, 0.35, 0.40, 0.45]
):
    strategy = TradingStrategy()
    # Modify strategy with params...
    
    backtest = Backtest(strategy)
    backtest.run(prices)
    
    results = backtest.get_results()
    results_list.append({
        'buy': buy_thresh,
        'sell': sell_thresh,
        'sharpe': results['sharpe_ratio'],
        'pnl': results['total_pnl']
    })

# Find best
best = max(results_list, key=lambda x: x['sharpe'])
print(f"Best params: {best}")
```
